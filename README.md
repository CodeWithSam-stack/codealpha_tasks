# Python Projects Collection

This repository contains a collection of beginner-friendly Python projects developed to strengthen programming fundamentals, problem-solving skills, file handling techniques, and user interaction through command-line applications.

## 📌 Projects Included

### 🎮 1. Hangman Game

A console-based implementation of the classic Hangman word guessing game. The program randomly selects a word, and the player must guess it one letter at a time before running out of attempts. The game provides real-time feedback for correct and incorrect guesses while tracking previously entered letters.

**Key Features:**

* Random word selection
* Input validation
* Limited guessing attempts
* Win/Loss detection
* Interactive gameplay experience

---

### 📈 2. Stock Portfolio Tracker

A simple portfolio management application that allows users to track stock investments and calculate the total value of their portfolio. Users can select stocks, enter quantities, generate investment summaries, and save portfolio data to a text file for future reference.

**Key Features:**

* Predefined stock price database
* Portfolio value calculation
* Investment summary generation
* File handling for saving results
* User-friendly command-line interface

---

### 📧 3. Email Address Extractor

A file-processing utility that scans text files and extracts all valid email addresses using Regular Expressions (Regex). The application removes duplicate entries, displays the extracted emails, and allows users to save the results in TXT format, CSV format, or both.

**Key Features:**

* Email extraction using Regex
* Duplicate email removal
* TXT and CSV export support
* File existence validation
* Error handling and user-friendly output
* Sorted email list generation

---

### 🤖 4. Simple Chatbot

A rule-based chatbot that communicates with users through predefined responses. It recognizes common greetings, questions, and conversational keywords while providing randomized replies to create a more engaging interaction experience.

**Key Features:**

* Keyword-based response system
* Multiple randomized replies
* Continuous conversation loop
* Exit commands for ending chats
* Beginner-friendly chatbot implementation
